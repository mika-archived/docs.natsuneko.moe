Animation Auto Assignment は、現在開いているアニメーションクリップに対して、各オブジェクトの変更を追従し、ヒエラルキー上の変更にあわせてアニメーションクリップの内容を変更する Unity 拡張機能です。  
これを使うことで、ヒエラルキー内の移動や名前変更などによる Missing Property を防ぐことが出来ます。
