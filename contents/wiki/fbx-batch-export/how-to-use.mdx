Blender の Layout View や Modeling View などのサイドバーから、 FBX Batch Export タブを開きます。  
開いた後、 `Generic` セクションの `Export Directory` に出力先ディレクトリを設定します。

設定後、 3D View から出力したいモデルを選択状態にし、 `Generate` ボタンを押下することで出力が行われます。

## 3D モデルの設定について

`Export Settings` の項目から、出力ページの設定と同様の設定が可能です。  
必要に応じてご利用ください。
