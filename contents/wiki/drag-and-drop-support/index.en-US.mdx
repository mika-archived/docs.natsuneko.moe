Drag and Drop Support is a Blender add-on and peripheral tools to enable drag-and-drop file imports in Blender.

<Alert title="Warning" type="warning">
  This tool is experiemntal. Use it at your own risk as it may cause unintended
  behavior or crash Blender.
</Alert>

The following file formats are supported for drag-and-drop import:

- FBX (`*.fbx`)
- glTF (`*.glb`, `*.gltf`)
- OBJ (`*.obj`)
- X3D (`*.x3d`)
