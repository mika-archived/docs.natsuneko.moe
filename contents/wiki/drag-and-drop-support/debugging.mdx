Drag and Drop Support では環境固有の問題が起こることがあるため、デバッグ用のコンソール出力があります。  
以下の方法でツールを起動することで行えます。

1. Blender を通常通り起動する
2. タスクマネージャーから `blender.exe` のプロセス ID を確認する
3. コマンドプロンプトを開く
4. コマンドプロンプト上で、ダウンロードしたファイルを解凍したところまで移動する
   - 例: `cd C:/Users/natsuneko/Downloads`
5. コマンドプロンプトから、 `./blender-hook.exe <Blender のプロセス ID>` を実行する
   - 例： `./blender-hook.exe 12345`
6. Blender で[使用方法](./how-to-use) に従った操作を行う
7. Drag-and-Drop を行う
8. Blender を終了する

終了後、センシティブな情報 (例: パス名) を除いたテキストを作者の Discord や GitHub の Issues に書き込んでください。  
また、使用しているセキュリティソフトがありましたら、それもお伝えください。
