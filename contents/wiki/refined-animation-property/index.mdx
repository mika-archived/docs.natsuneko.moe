Refined Animation Property は、 Unity のデフォルトのアニメーション作成機能を改善し、作業効率を上げるためのツールです。  
導入すると、以下の機能が修正・追加されます。

1. ポップアップメニューから、連続して Animation Property を追加できるようになります
2. Animation Property を追加する際、検索することが出来るようになります
3. 内部での ID 生成アルゴリズムを修正し、エラーの発生を起こりにくくします
4. Easing Functions に応じたアニメーションカーブを設定できるようになります
