ここでは、基本的な設定を行って、実際にテキストメッシュを出力するまでを説明します。  
各設定項目についての詳細は、 [設定一覧](./text-mesh-creator/preferences/) を参照してください。

まずはじめに、 Blender の Layout View サイドバーから、 TextMesh Creator のタブを開きます。  
開いたら、「Strings」に対応するテキストボックスに、生成して欲しい文字もしくは文字列を入力してください。  
文字列の長さに制限はありませんが、センテンス毎などに設定することを推奨します。

次に、「Fonts」から、文字を描画するフォントを選択してください。  
フォントファイルは OTF, TTF などの形式に対応しています。

最後に、「Export Directory」から、出力先のディレクトリを選択してください。  
生成されるすべてのファイルは、このディレクトリ直下に生成されます。

あとは、「Generate」をクリックすることで生成されます。
